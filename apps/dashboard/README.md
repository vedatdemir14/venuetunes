# apps/dashboard — Mekan Yönetim Paneli (React + Vite)

Mekan sahibinin Spotify hesabını bağladığı, kuyruğu ve sohbeti yönettiği panel.

## Planlanan Özellikler

- Spotify OAuth bağlama + Connect cihaz seçimi
- Kuyruk: veto, öne alma, kara liste
- Sohbet moderasyonu: gizle, sustur, engelle
- QR kod üretimi (masa bazlı)
- (Opsiyonel) Web Playback SDK ile panel üzerinden çalma

## Kurulum (Faz 4'te doldurulacak)

```bash
pnpm install
pnpm dev
```
